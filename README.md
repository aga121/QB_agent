## 简介

基于 Claude 智能体为内核打造的 Web 版智能体平台，性能强劲、任务完成效率极高，面向企业复杂办公场景具备极强的兼容性。硬件成本做到极低，16GB 内存可同时支撑数千人使用，非常适合中小企业：不愿投入昂贵硬件但又需要一个贯穿业务的智能中枢。系统采用 Linux 内核的 UID/GID 与 cgroup2，结合算法并围绕 Bash 包裹机制，实现低于 1MB 内存的沙箱隔离，成本低廉且安全稳定。算力方面在非私有化部署下，采用 GLM Coding 驱动模型，在保障智能体能力的同时，将硬件与算力成本压到 1000 人规模企业仅需每月几百到几千元。内置 agent 智能回收机制、算法优化及巧妙的沙箱设计，使 1000 商业用户/企业的硬件成本低至几百元，算力开销也极为节省，可支撑日常工作。这是非常令人震惊的。
